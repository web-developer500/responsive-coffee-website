☕ Responsive Coffee Website

A modern, responsive coffee shop website built with HTML5 and CSS3. The project focuses on creating a clean UI, smooth layout, and a fully responsive experience across different screen sizes.



✨ Features
📱 Fully Responsive Design
🎨 Modern & Clean UI
☕ Hero Section
👤 About Section
📋 Menu Section
💬 Testimonials Section
📞 Contact Section
🔗 Footer with Social Links
⚡ Smooth User Experience
💻 Beginner-Friendly Code Structure



🛠️ Built With
HTML5
CSS3
Google Fonts
